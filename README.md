# Lab 1 - Grade Evaluator & Archiver

## Files Included
- grade-evaluator.py
- organizer.sh
- grades.csv

## Description
This project reads student grades from a CSV file, validates the scores and weights, calculates the final grade and GPA, determines pass/fail status, and identifies formative assignments eligible for resubmission.

The shell script archives the old grades.csv file with a timestamp, creates a fresh empty CSV file, and logs the archive activity.

## How to Run the Python Program
Run:
```bash
python3 grade-evaluator.py
